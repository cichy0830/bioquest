# U23 behavior_sensing assets

本資料夾目前不放待審圖。U23 視覺尚未由使用者核准，因此 runtime 只使用 CSS/HTML fallback，避免 404 或誤接未核准素材。

正式素材核准後建議 hook：

- `behavior-sensing-entry-wide.webp`：一般頁純背景。
- `behavior-sensing-briefing-azhe-wide.webp`：簡報頁 safe scene。
- `behavior-sensing-briefing-azhe-mobile.webp`：若視覺線另提供手機版，再接入 `<picture>`。
- `behavior-sensing-prep-owl.webp`：準備頁單元專屬貓頭鷹。
- `behavior-sensing-report-owl.webp`：若需要單元專屬回報貓頭鷹，需先由視覺線核准。

15 枚正式徽章預留於 `../shared-assets/badges/behavior_sensing/`。圖檔落地前，收藏牆顯示中性「正式徽章素材待接」狀態並保留亮燈/灰階條件，不使用幾何徽章 placeholder。
